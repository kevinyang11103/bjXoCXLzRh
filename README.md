# 项目简介 python611

本项目是基于Python深度学习的电影评论情感分析系统。

## 技术栈

- **开发工具**：IDEA、Eclipse
- **编程语言**：Python
- **数据库**：MySQL 5.7+
- **框架**：Flask、Django
- **算法**：深度学习、卷积神经网络、Word2vec

## 功能模块

- 首页
- 电影评价
- 个人信息
- 用户管理

## 系统角色

- 用户：浏览电影评论，获取情感分析结果。
- 管理员：管理系统用户及内容。

## 运行环境

- Python环境
- MySQL数据库环境

## 部署步骤

1. 环境配置：安装Python、MySQL及必要库。
2. 数据库搭建：根据schema文件创建数据库和表。
3. 程序部署：配置服务器，运行程序。

## 目录结构

```
/
├── app           # 应用代码
│   ├── models    # 数据模型
│   ├── views     # 视图逻辑
│   └── templates # 前端模板
├── static        # 静态文件
├── migrations    # 数据库迁移脚本
├── requirements.txt  # 依赖列表
└── manage.py     # 管理脚本
```
